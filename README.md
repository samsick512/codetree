# 🌱 Python 코딩 실력 기르기 (CodeTree)

CodeTree를 활용하여 Python 코딩 실력을 꾸준히 향상시키는 과정을 기록하는 레포지토리입니다.  
기초 문법부터 알고리즘 문제 해결 능력까지 단계적으로 성장하는 것을 목표로 합니다.

---

## 🎯 목표
- Python 기초 문법 완벽 이해
- 문제 해결 능력(PS) 향상
- 알고리즘 사고력 강화
- 꾸준한 코딩 습관 형성

---

## 🧩 학습 내용
- 함수
- 정렬
- 시뮬레이션
- 완전탐색
- 케이스별로 나누기
- Ad-Hoc
  
---

## 🛠 사용 언어
- Python 🐍

---


---

## 📈 진행 방식
- CodeTree 문제 풀이
- 하루 최소 1문제 이상 풀이
- 풀이 코드 및 설명 정리
- 꾸준한 커밋으로 성장 기록

---

## 🚀 목표 수준
- 기본 문제 → 빠르고 정확하게 해결
- 중급 문제 → 스스로 해결 가능
- 알고리즘 문제 → 논리적으로 접근

---

## 💡 한마디
> 꾸준함이 실력을 만든다. 🔥
